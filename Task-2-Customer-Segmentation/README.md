# Task 2 - Customer Segmentation Project

## Objective
Perform customer segmentation based on demographics and spending behavior to generate actionable business insights.

## Tools Used
- Power BI
- Excel
- Python (Optional)

## Features
- Total Customers KPI
- Average Spending Score
- Average Annual Income
- Customer Segment Distribution
- Income vs Spending Analysis
- Segment-wise Income and Spending Comparison
- Gender Distribution Analysis
- Interactive Filters and Slicers

## Key Insights
- Identified 5 distinct customer segments.
- Analyzed customer purchasing behavior and preferences.
- Visualized high-value and low-value customer groups.
- Enabled data-driven customer targeting strategies.

## Learning Outcomes
- Customer Analytics
- Customer Segmentation
- Data Visualization
- Business Intelligence
- Interactive Dashboard Design
